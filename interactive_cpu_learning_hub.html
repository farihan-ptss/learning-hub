Silibus 1.5.1

Apakah itu Central Processing Unit (CPU)?

Central Processing Unit (CPU) ialah otak komputer yang bertanggungjawab memproses semua arahan, membuat pengiraan, dan mengawal operasi semua komponen komputer.

Ia menerima data daripada ingatan (Memory) atau peranti input (seperti papan kekunci), memproses data tersebut, dan menghantar hasilnya semula ke ingatan atau peranti output (seperti monitor).

Memahami Isyarat "Bus" (Sistem Penghubung)

Data Bus

Membawa bit-bit data sebenar di antara CPU, memori, dan peranti luaran.

Address Bus

Membawa alamat memori (lokasi penyimpanan) di mana data tersebut diletakkan atau ingin dibaca.

Control Bus

Membawa isyarat kawalan (seperti arahan membaca 'Read' atau menulis 'Write') daripada Control Unit.

Silibus 1.5.4

Apakah itu Kitaran Arahan (Instruction Cycle)?

Kitaran arahan ialah langkah berturut-turut yang sentiasa diulangi oleh CPU untuk mengambil, memahami, dan melaksanakan arahan pengaturcaraan daripada memori utama.

Fasa 1: FETCH

📖 Fetch Cycle

Kata Kunci: Fetch = Ambil

CPU mengambil arahan mentah daripada memori RAM. CPU belum memahami arahan tersebut pada tahap ini.

Analogi: 📚 Mengambil buku resipi makanan daripada rak buku.
Fasa 2: DECODE

🤔 Decode Cycle

Kata Kunci: Decode = Faham

Control Unit (CU) mentafsir dan menterjemah kod arahan tersebut untuk mengetahui tugas yang patut dilakukan.

Analogi: 📝 Membaca resipi dan memahami keperluan bahan memasak.
Fasa 3: EXECUTE

☕ Execute Cycle

Kata Kunci: Execute = Laksana

CPU menjalankan arahan yang telah difahami. ALU dipanggil untuk membuat pengiraan jika melibatkan operasi logik/aritmetik.

Analogi: 🍳 Memasak hidangan di dapur berdasarkan resipi tadi.
🎓

Bagaimana Mengingat Untuk Peperiksaan?

Bayangkan tugasan soalan daripada Pensyarah:
Fetch 📄 → Ambil helaian kertas soalan.
Decode 🤔 → Baca dan faham kehendak soalan.
Execute ✍️ → Menjawab soalan secara amali.
